表面

大氣層: 主要成分是氫（83%）和氦（15%），還有約2.3%的甲烷，讓它呈現藍綠色。

冰巨星特性: 不同於木星、土星，天王星上層大氣下是高壓下的水、氨、甲烷混合物，沒有金屬氫，因此被歸類為冰巨星。

無明確固體表面: 氣體大氣層和內部的「冰」海洋之間沒有分界，只有當氣壓達到1巴（100 kPa）的深度，被定義為「表面」。

雲層與風暴: 雖然早期觀測認為它平靜，但哈伯望遠鏡觀測顯示有雲層和風暴活動，風速可達每秒250公尺。
