氣體與液體: 土星主要由氫氣(96.3%) 和氦氣(3.25%) 構成，內部深處氣體壓力極高，行為類似液體金屬，但沒有可觸碰的固體表面。

雲層: 外部有氨冰晶、硫化氫銨或水組成的雲層，形成黯淡的條紋和風暴。

核心: 內部有緻密的核心，包含氫、氦、冰和岩石的混合物，而不是地球那種層狀結構。
