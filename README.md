【星宇・探索】互動式宇宙科普平台 (Interactive Space Explorer)
🌌 專案簡介
本專案是一個基於原生 Web 技術開發的互動式天文科普平台。使用者可以透過直觀的介面探索太陽系、仙女座星系，並查詢黃道十二宮的相關資訊。本作品旨在將抽象的天文數據轉化為具備美感與互動性的數位體驗。

🛠️ 技術棧 (Tech Stack)
前端開發：HTML5, CSS3, JavaScript (Vanilla JS)

介面設計：獨立完成 UI/UX 設計，實作深色模式 (Dark Mode) 與多層次佈局

開發工具：VS Code, AI 輔助開發 (ChatGPT / Claude)

🤖 AI 協作開發聲明 (AI-Augmented Development)
在本專案開發過程中，我實踐了「人機協作」的現代開發流程：

邏輯優化：運用 AI 協助生成星球運動的數學演算邏輯，並由人工進行代碼重構與功能驗證。

CSS 效能精煉：藉由 AI 指令優化 CSS 渲染效能與深色模式的色階對比，確保在各裝置上的視覺一致性。

除錯與測試：利用 AI 輔助進行 JavaScript 邏輯除錯，縮短約 60% 的開發週期，並透過人工主導最終的功能測試與約化。

🌟 核心功能
星系跳轉系統：實作太陽系與仙女座星系的動態導覽跳轉。

行星成員全覽：彙整八大行星科學數據，以卡片式佈局呈現細節。

星座索引系統：整合黃道十二宮資訊，提供直觀的圖像化索引。

即時太空動態：結合動態消息欄位，展示天文新知。

🔒 隱私聲明
為保護個人隱私，本專案原始碼中之聯絡資訊（如電話）已進行脫敏處理 (Masking)。
